Terminal use:
0. rename the main index.html file to index.html in order for the next steps to work!
1. npm init --yes
2. npm install --save-dev lite-server
3. npx lite-server
4. Go to to package.json file, open it and add in "scripts" section the following: 
"start": "lite-server",

5. If the name in the top in the json file is underlined, edit it with lower case letters!
6. npm start
7. npm install handlebars --save
8. npm start
9. Go to index.html and add a script in the head:
<script src="./node_modules/handlebars/dist/handlebars.min.js"></script>
